# learn-langgraph
Learn about Langgraph Library 

## Setup

### Virtual Environment
A Python virtual environment `myenv` has been created for this project.

**Activate the virtual environment:**
```bash
source myenv/bin/activate
```

**Deactivate when done:**
```bash
deactivate
```
